HTTP API Reference¶

For the examples, we’ll use httpie (can be installed with pip).

$ mkdir ~/.httpie; echo '{"default_options":["--session=default"]}' > ~/.httpie/config.json
$ pip install httpie
Collecting httpie
[...]
Successfully installed httpie-0.9.9
POST /user/login¶

Generate a JWT token if the provided credentials are valid.

Example request:

$ http POST :8081/user/login username=marco password=<redacted>

Example response:

HTTP/1.1 200 OK
Content-Length: 210
Content-Type: application/json; charset=utf-8
Date: Wed, 18 Jan 2017 14:44:02 GMT
Server: Python/3.5 aiohttp/1.2.0

{
    "token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJpc3MiOiJhdHMuYXV0aCIsInN1YiI6Im1hcmNvIiwiZXhwIjoxNDg0ODM3MDQyLjg3NTYzMDEsImlhdCI6MTQ4NDc1MDY0Mi44NzU2MzAxfQ.DqBYOAIWuCp2wm6KwEdwrsodYmBK3f9fvQNgqXcTfSE"
}
JSON Parameters:
 
  • username (string) – the username to authenticate
  • password (string) – the password to authenticate
Response Headers:
 
  • Content-Type – always application/json
Status Codes:
  • 200 OK – a new token has been issued
  • 401 Unauthorized – the credentials are not valid
POST /user/logout¶

Revoke an authentication token.

Example request:

$ http POST :8081/user/logout 'Authorization: Bearer eyJ0eXAiOiJKV1[...]sodYmBK3f9fvQNgqXcTfSE'

Example response:

HTTP/1.1 204 No Content
Content-Length: 0
Content-Type: application/octet-stream
Date: Wed, 18 Jan 2017 14:48:39 GMT
Server: Python/3.5 aiohttp/1.2.0
Request Headers:
 
  • Authorization – token to revoke
Response Headers:
 
  • Content-Type – always application/octet-stream
Status Codes:
  • 204 No Content – the token has been blacklisted
POST /user/logout¶

Revoke all user tokens.

Example request:

$ http POST :8081/user/logout_all 'X-Auth-UserId: marco'

Example response:

HTTP/1.1 204 No Content
Content-Length: 0
Content-Type: application/octet-stream
Date: Wed, 18 Jan 2017 15:04:11 GMT
Server: Python/3.5 aiohttp/1.2.0
Request Headers:
 
  • X-Auth-UserId – user that needs to revoke tokens
Response Headers:
 
  • Content-Type – always application/octet-stream
Status Codes:
  • 204 No Content – the tokens have been blacklisted
